# GeoInfoSystem Connector (QGIS Plugin)

Профессиональный плагин для QGIS, обеспечивающий полную интеграцию с микросервисной платформой **GeoInfoSystem**. Плагин позволяет работать с геоданными системы напрямую из настольного приложения, поддерживая иерархическую структуру проектов, визуализацию рельефа и двустороннюю синхронизацию векторов.

## Основные возможности

*   **OAuth2 Авторизация:** Безопасный вход через системный браузер (SSO) с использованием протокола PKCE.
*   **Иерархический проводник:** Полная поддержка вложенности папок и объектов, идентичная веб-интерфейсу системы.
*   **Двусторонняя синхронизация векторов:** 
    *   Загрузка точек, линий и полигонов напрямую из `geodata-service`.
    *   Отправка новых объектов и изменений геометрии/атрибутов обратно на сервер одной кнопкой (**Synchronize Changes**).
    *   Автоматическое получение серверных ID для новых объектов.
*   **Поддержка 3D-рельефа:** Подключение слоев рельефа (Quantized Mesh) через нативный провайдер QGIS Mesh Layer.
*   **WMS интеграция:** Автоматическое формирование строк подключения для растровых слоев из GeoServer.
*   **Zero Trust & Security:** Работа исключительно через API Gateway (порт 80) и SSO (sso.localhost), отсутствие прямого доступа к БД.

## Архитектура плагина

```
pyqgis-plugin/
├── core/
│   ├── auth.py          # Логика OAuth2 PKCE, временный HTTP-сервер для Callback
│   ├── api_client.py    # REST-клиент для взаимодействия с микросервисами
│   └── layer_factory.py # Фабрика создания слоев (Vector, WMS, Terrain Mesh)
├── ui/
│   └── dock_widget.py   # Панель проводника с поддержкой дерева папок и синхронизации
├── main.py              # Точка входа, диагностика провайдеров и жизненный цикл
└── metadata.txt         # Метаданные расширения QGIS
```

## Установка

1.  Скопируйте папку `pyqgis-plugin` в директорию профиля QGIS:
    *   **Windows:** `%APPDATA%\QGIS\QGIS3\profiles\default\python\plugins`
    *   **Linux:** `~/.local/share/QGIS/QGIS3/profiles/default/python/plugins`
2.  Перезапустите QGIS.
3.  Активируйте в меню: `Модули -> Управление модулями -> GeoInfoSystem Connector`.

## Использование

1.  Нажмите иконку **GeoInfoSystem Connector**. Пройдите авторизацию в браузере.
2.  В появившейся панели разверните проект. Вы увидите три группы:
    *   **Vectors:** Вложенные папки и векторные объекты.
    *   **WMS Layers:** Растровые слои.
    *   **Terrain Layers:** Слои рельефа (рекомендуется использовать в 3D Map View).
3.  **Синхронизация:** 
    *   Добавьте векторный слой, перейдите в режим редактирования QGIS, внесите изменения или добавьте новые объекты.
    *   Нажмите **"Synchronize Changes"** для сохранения данных в облако.

## Диагностика и отладка

Плагин ведет детальный лог всех запросов и ответов. Чтобы увидеть его:
1.  Нажмите на иконку облачка/текста в нижнем правом углу QGIS.
2.  Перейдите на вкладку **GeoInfoSystem**.
3.  Там отображаются URL запросов, тела ответов и список доступных в системе провайдеров.

## Требования

*   **QGIS:** 3.18 или выше (рекомендуется 3.34+ LTR для корректной работы Mesh/Terrain).
*   **Окружение:** Прописанный `sso.localhost` в файле `hosts` вашей ОС.
