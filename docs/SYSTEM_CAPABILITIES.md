# 🌍 ГеоИнфоСистема: Единый реестр возможностей

**ГеоИнфоСистема** — это высокотехнологичная микросервисная платформа для комплексного управления геопространственными данными, объединяющая классическую картографию, спутниковый мониторинг, 3D-визуализацию рельефа и систему электронного документооборота.

---

## 1. Картография и визуализация (2D/3D)
*   **Двухрежимный интерфейс:**
    *   **2D Карта (OpenLayers):** Работа с векторными слоями, WMS/WMTS подложками и растрами.
    *   **3D Глобус (CesiumJS):** Визуализация рельефа и объектов в трехмерном пространстве.
*   **Высокопроизводительный рендеринг (MVT):**
    *   Использование **Mapbox Vector Tiles** для мгновенного отображения слоев с десятками тысяч объектов.
    *   Интеграция с **pg_tileserv** для потоковой передачи векторов напрямую из PostGIS.
    *   Гибридная модель: автоматическое переключение между MVT (для просмотра) и GeoJSON (для редактирования).
*   **Управление слоями:** Переключение базовых карт, регулировка прозрачности, иерархическое дерево слоев с неограниченной вложенностью папок.
*   **Векторные объекты:** Поддержка точек (маркеров), линий (мультилиний) и полигонов.
*   **Стилизация KML/KMZ:** Сохранение и отображение пользовательских стилей (цвета, толщина линий, кастомные иконки), импортированных из KML/KMZ файлов.
*   **Инструмент сравнения «Шторка» (Swipe Tool):** Визуальное сравнение двух растровых слоев в реальном времени с автоматическим приближением к границам выбранных снимков.
*   **Инструмент «Значение растра» (Raster Value Tool):** Быстрое интерактивное определение точного значения пикселя активного растрового слоя (как статических Imagery-слоев проекта, так и динамических Staging-слоев геоанализа) в точке клика на 2D-карте или 3D-глобусе с выводом координат и значений через API TiTiler.
*   **Умная линейка и площадь:** Инструменты для высокоточного измерения расстояний и площадей на карте с поддержкой различных единиц измерения и автоматическим расчетом в режиме реального времени.
*   **Интерактивная карточка объекта (ObjectDetails):** Полнофункциональная панель управления каждым элементом на карте:
    *   **Управление метаданными:** Просмотр и редактирование атрибутов (название, описание, статус, папки).
    *   **Геометрический анализ:** Отображение списка координат всех вершин, расчет длин и площадей.
    *   **Интерактивное редактирование:** Режим изменения узлов геометрии и массовая корректировка высоты (Altitude Offset).
    *   **Динамическая стилизация:** Кастомизация внешнего вида (цвет, заливка, толщина линий) и загрузка пользовательских иконок для точек.
    *   **Мультимедиа и документы:** Прямая привязка файлов (PDF, Office) и просмотр живых видеотрансляций с IP-камер.

## 2. Анализ данных дистанционного зондирования (ДЗЗ)
*   **Поддержка спутников:** Автоматизированный импорт и обработка данных **Sentinel-2** (.SAFE) и **Landsat 8** (.tar.gz/zip).
*   **Автоматическое извлечение границ (BBox):** При обработке любого растра система автоматически вычисляет его точные пространственные границы (Bounding Box) с помощью GDAL и сохраняет их в базе данных. Это позволяет выполнять быстрый поиск и навигацию по слоям.
*   **Спектральный анализ:** Автоматический расчет более 15 индексов:
    *   **Растительность:** NDVI, EVI, SAVI, GNDVI (здоровье посевов, хлорофилл).
    *   **Водные ресурсы:** NDWI, NDMI (влажность почв, границы водоемов).
    *   **Инфраструктура:** NDBI (застройка), NBR (анализ гарей и пожаров).
    *   **Криосфера:** NDSI (детектирование снега и льда).
*   **RGB-комбинации:** Визуализация в различных режимах (Natural Color, False Color, Agriculture, Urban и др.).
*   **Оптимизация COG:** Конвертация тяжелых снимков в Cloud Optimized GeoTIFF для мгновенной отрисовки без задержек.

## 3. 3D-рельеф и Terrain
*   **Terrain Pipeline:** Генерация 3D-рельефа (Cesium Quantized-Mesh) из GeoTIFF DEM.
*   **Self-hosted инфраструктура:** Полная независимость от Cesium ion — генерация и раздача тайлов рельефа происходит на собственных мощностях.
*   **Асинхронная обработка:** Мощные Python-воркеры для фоновой подготовки тайловых деревьев.

## 4. Продвинутая гео-аналитика (Geo-Analysis)
*   **Управляемый схемами интерфейс (Schema-Driven UI):**
    *   **Динамическая генерация форм:** Генерация диалоговых окон запуска во фронтенде (компонент `DynamicAnalysisDialog.vue`) на основе декларативных JSON-схем (Draft-07), поставляемых плагинами воркера.
    *   **Специализированные ГИС-виджеты:** Автоматическая интеграция полей формы со сложными элементами ввода: выбор слоев, прикрепленные документы проекта, интерактивное указание точек на карте, а также визуальные редакторы таблиц (матрицы).
    *   **Двухсторонняя валидация:** Проверка параметров по JSON-схемам как на стороне UI, так и на бэкенде с помощью строгого валидатора перед отправкой задач на исполнение.
*   **Изолированная среда вычислений:** Выполнение тяжелых пространственных операций в выделенном воркере с использованием RAM-диска (`tmpfs`) для максимальной скорости.
*   **Анализ рельефа:**
    *   **Генерация изолиний (Contours):** Создание 2D и **3D-векторных линий** равных высот из DEM-растров с настраиваемым шагом.
    *   **Деривативы рельефа:** Расчет крутизны склонов (Slope), экспозиции (Aspect) и теневой отмывки (Hillshade) с автоматическим масштабированием единиц измерения для географических координат (градусы в метры).
*   **Пространственная статистика:**
    *   **Зональная статистика (Zonal Stats):** Расчет статистических показателей растра (мин, макс, среднее) в границах векторных полигонов.
*   **Векторные операции и Импорт:**
    *   **Импорт DXF:** Конвертация слоев DXF в плоский или 3D GeoJSON с использованием `gdal.VectorTranslate` (опция `dim="XYZ"` для сохранения высот/глубин) с валидацией координатной привязки (ручной ввод CRS источника при ее отсутствии в файле).
    *   **Пространственные функции:** Автоматическое создание буферных зон, поиск пространственных пересечений и агрегация данных.
*   **Гидрологическое моделирование:** Выделение водосборных бассейнов и сетей водотоков на основе анализа стоков по ЦМР.

## 5. Управление документами и OnlyOffice
*   **Свойства проекта:** Модальное окно просмотра и редактирования свойств проекта на 2D и 3D картах. Включает изменение названия/описания, получение и сжатие основного изображения проекта, отображение охватывающего bbox и единый реестр прикрепленных к проекту документов.
*   **Привязка к объектам:** Возможность прикрепить неограниченное количество документов (PDF, DOCX, XLSX, изображения) к любому объекту на карте.
*   **Онлайн-редактирование:** Полная интеграция с **OnlyOffice**. Редактирование офисных файлов прямо в браузере с автоматическим сохранением версий в S3-хранилище (MinIO).
*   **Полнотекстовый поиск:** Поиск по содержимому документов (включая PDF и офисные форматы) через Elasticsearch.

## 5. Мониторинг и Видеостриминг
*   **Интеграция IP-камер:** Поддержка камер (например, Hikvision) с выводом потока на карту.
*   **Динамический стриминг:** Использование **MediaMTX** для преобразования RTSP-потоков в HLS для просмотра в браузере.
*   **Управление потоками:** Запуск трансляции только при активном просмотре для экономии ресурсов сервера и трафика.

## 6. Импорт, Экспорт и Печать карт
*   **Direct Upload:** Механизм загрузки файлов до 10 ГБ напрямую в S3 через Presigned URLs, минуя перегрузку API Gateway.
*   **KML/KMZ Интеграция:** Полноценный импорт геометрии, метаданных и вложенных ресурсов (иконок, стилей).
*   **Асинхронная печать отчетов (GeoPrint):** Формирование высококачественных картографических PDF-отчетов форматов А4-А0 (книжная/альбомная ориентация) с интерактивным видоискателем (Shot Frame) на карте для точного кадрирования. Полезная площадь карты в отчете увеличена до 80% страницы, внешние поля уменьшены до 0.5 см, а размеры Matplotlib-холста рассчитываются динамически для исключения пропорциональных искажений. Интегрировано автоматическое объединение растров (через TiTiler) и векторных слоев (с заливкой только полигональных объектов).

## 7. Интеграция с QGIS (Настольный клиент)
*   **Тяжелый аналитический клиент:** Расширение возможностей веб-платформы инструментами профессиональной настольной ГИС.
*   **OAuth2/PKCE Авторизация:** Безопасный вход через SSO с использованием системного менеджера паролей QGIS.
*   **Двусторонняя синхронизация:** Редактирование векторных слоев (точки, линии, полигоны) в QGIS с мгновенным сохранением изменений в `geodata-service`.
*   **Растровые слои XYZ Tiles:** Прямая загрузка растровых слоев через TiTiler в QGIS в виде слоев XYZ Tiles (без принудительных стилей для возможности гибкой стилизации на стороне QGIS).
*   **Потоковая обработка растров (COG):** Работа с высокодетальными спутниковыми снимками и матрицами высот без необходимости их скачивания на локальный диск.
*   **3D Mesh Terrain:** Визуализация и анализ 3D-рельефа системы в нативном 3D-окне QGIS.
*   **Иерархический проводник:** Полный доступ к структуре папок, проектов, XYZ Tiles слоев и библиотек рельефа через панель плагина.

## 8. Безопасность и Администрирование
*   **Ролевая модель (RBAC):** Роли ADMIN, EDITOR, VIEWER с четким разграничением прав на просмотр и редактирование.
*   **Безопасный доступ:** Аутентификация через JWT, верификация email, восстановление пароля через OTP.
*   **Изоляция данных:** Модель "Database per Microservice" обеспечивает высокую надежность и независимость сервисов.

## 9. Технологический фундамент
*   **Бэкенд:** Java 17, Spring Boot 3.2+, Spring Cloud, Kafka, GeoTools 28.2, openhtmltopdf.
*   **Фронтенд:** Vue.js 3, Vite, TypeScript, Vuetify 3, OpenLayers (MVT), CesiumJS.
*   **Хранилища:** PostgreSQL/PostGIS, Elasticsearch, MinIO, Redis.
*   **ГИС-серверы:** 
    *   **TiTiler:** Динамическая публикация растровых слоев (COG) напрямую из S3 по протоколу XYZ.
    *   **pg_tileserv:** Динамическая публикация векторных тайлов (MVT).
*   **Инфраструктура:** Docker, Docker Compose, Nginx (кэширование MVT и HLS), MediaMTX.
