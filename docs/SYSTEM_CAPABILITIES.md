# 🌍 ГеоИнфоСистема: Единый реестр возможностей

**ГеоИнфоСистема** — это высокотехнологичная микросервисная платформа для комплексного управления геопространственными данными, объединяющая классическую картографию, спутниковый мониторинг, 3D-визуализацию рельефа и систему электронного документооборота.

---

## 1. Картография и визуализация (2D/3D)
*   **Двухрежимный интерфейс:**
    *   **2D Карта (OpenLayers):** Работа с векторными слоями, WMS/WMTS подложками и растрами.
    *   **3D Глобус (CesiumJS):** Визуализация рельефа и объектов в трехмерном пространстве.
*   **Управление слоями:** Переключение базовых карт, регулировка прозрачности, иерархическое дерево слоев с неограниченной вложенностью папок.
*   **Векторные объекты:** Поддержка точек (маркеров), линий (мультилиний) и полигонов.
*   **Стилизация KML/KMZ:** Сохранение и отображение пользовательских стилей (цвета, толщина линий, кастомные иконки), импортированных из KML/KMZ файлов.

## 2. Анализ данных дистанционного зондирования (ДЗЗ)
*   **Поддержка спутников:** Автоматизированный импорт и обработка данных **Sentinel-2** (.SAFE) и **Landsat 8** (.tar.gz/zip).
*   **Спектральный анализ:** Автоматический расчет более 15 индексов:
    *   **Растительность:** NDVI, EVI, SAVI, GNDVI (здоровье посевов, хлорофилл).
    *   **Водные ресурсы:** NDWI, NDMI (влажность почв, границы водоемов).
    *   **Инфраструктура:** NDBI (застройка), NBR (анализ гарей и пожаров).
    *   **Криосфера:** NDSI (детектирование снега и льда).
*   **RGB-комбинации:** Визуализация в различных режимах (Natural Color, False Color, Agriculture, Urban и др.).
*   **Оптимизация COG:** Конвертация тяжелых снимков в Cloud Optimized GeoTIFF для мгновенной отрисовки без задержек.

## 3. 3D-рельеф и Terrain
*   **Terrain Pipeline:** Генерация 3D-рельефа (Cesium Quantized-Mesh) из GeoTIFF DEM.
*   **Self-hosted инфраструктура:** Полная независимость от Cesium ion — генерация и раздача тайлов рельефа происходит на собственных мощностях.
*   **Асинхронная обработка:** Мощные Python-воркеры для фоновой подготовки тайловых деревьев.

## 4. Управление документами и OnlyOffice
*   **Привязка к объектам:** Возможность прикрепить неограниченное количество документов (PDF, DOCX, XLSX, изображения) к любому объекту на карте.
*   **Онлайн-редактирование:** Полная интеграция с **OnlyOffice**. Редактирование офисных файлов прямо в браузере с автоматическим сохранением версий в S3-хранилище (MinIO).
*   **Полнотекстовый поиск:** Поиск по содержимому документов (включая PDF и офисные форматы) через Elasticsearch.

## 5. Мониторинг и Видеостриминг
*   **Интеграция IP-камер:** Поддержка камер (например, Hikvision) с выводом потока на карту.
*   **Динамический стриминг:** Использование **MediaMTX** для преобразования RTSP-потоков в HLS для просмотра в браузере.
*   **Управление потоками:** Запуск трансляции только при активном просмотре для экономии ресурсов сервера и трафика.

## 6. Импорт и Экспорт
*   **Direct Upload:** Механизм загрузки файлов до 10 ГБ напрямую в S3 через Presigned URLs, минуя перегрузку API Gateway.
*   **KML/KMZ Интеграция:** Полноценный импорт геометрии, метаданных и вложенных ресурсов (иконок, стилей).

## 7. Интеграция с QGIS (Настольный клиент) — *В процессе разработки*
*   **Тяжелый аналитический клиент:** Расширение возможностей веб-платформы инструментами профессиональной настольной ГИС.
*   **OAuth2/PKCE Авторизация:** Безопасный вход через SSO с использованием системного менеджера паролей QGIS.
*   **Двусторонняя синхронизация:** Редактирование векторных слоев (точки, линии, полигоны) в QGIS с мгновенным сохранением изменений в `geodata-service`.
*   **Потоковая обработка растров (COG):** Работа с высокодетальными спутниковыми снимками и матрицами высот без необходимости их скачивания на локальный диск.
*   **3D Mesh Terrain:** Визуализация и анализ 3D-рельефа системы в нативном 3D-окне QGIS.
*   **Иерархический проводник:** Полный доступ к структуре папок, проектов, WMS-слоев и библиотек рельефа через панель плагина.

## 8. Безопасность и Администрирование
*   **Ролевая модель (RBAC):** Роли ADMIN, EDITOR, VIEWER с четким разграничением прав на просмотр и редактирование.
*   **Безопасный доступ:** Аутентификация через JWT, верификация email, восстановление пароля через OTP.
*   **Изоляция данных:** Модель "Database per Microservice" обеспечивает высокую надежность и независимость сервисов.

## 9. Технологический фундамент
*   **Бэкенд:** Java 17, Spring Boot 3.2+, Spring Cloud, Kafka, GeoTools 28.2.
*   **Фронтенд:** Vue.js 3, Vite, TypeScript, OpenLayers, CesiumJS.
*   **Хранилища:** PostgreSQL/PostGIS, Elasticsearch, MinIO, Redis.
*   **ГИС-сервер:** GeoServer (публикация слоев, SLD-стили).
*   **Инфраструктура:** Docker, Docker Compose, Nginx (как прокси и кэш).
