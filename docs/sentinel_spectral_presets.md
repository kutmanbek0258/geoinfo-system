## 2. Часто используемые пресеты отображения

### Естественный цвет
**RGB:** `B4 - B3 - B2`  
Подходит для визуального просмотра, картографической подложки и общего анализа.

### False Color Vegetation
**RGB:** `B8 - B4 - B3`  
Растительность выглядит ярко-красной, удобно для оценки зеленой массы.

### Red Edge Agro
**RGB:** `B8A - B6 - B4`  
Хорошо показывает различия в состоянии растений и стресс.

### Moisture / Burn
**RGB:** `B12 - B11 - B4`  
Полезно для выявления сухих зон, пожаров и деградации.

### Urban / Soil
**RGB:** `B12 - B11 - B8`  
Удобно для анализа застройки, голой почвы и техногенных участков.

### Water emphasis
**RGB:** `B3 - B2 - B1` или `B8 - B3 - B2`  
Подходит для оценки водных объектов и прибрежных зон.

## 3. Часто используемые индексы

### NDVI — растительность
```text
NDVI = (B8 - B4) / (B8 + B4)

EVI — улучшенный индекс растительности
EVI = 2.5 * (B8 - B4) / (B8 + 6*B4 - 7.5*B2 + 1)

SAVI — индекс для редкой растительности и почвы
SAVI = (1 + L) * (B8 - B4) / (B8 + B4 + L)
Обычно: L = 0.5

GNDVI — хлорофилл и агроаналитика
GNDVI = (B8 - B3) / (B8 + B3)

NDMI — влажность растительности
NDMI = (B8 - B11) / (B8 + B11)

NBR — выгоревшие зоны
NBR = (B8 - B12) / (B8 + B12)

NDSI — снег и лёд
NDSI = (B3 - B11) / (B3 + B11)

NDBI — застройка
NDBI = (B11 - B8) / (B11 + B8)

NDWI — открытая вода
NDWI = (B3 - B8) / (B3 + B8)

```

### 4. Что использовать под разные задачи
```  
Растительность
B4, B8, B5, B6, B7, B8A
Индексы: NDVI, EVI, GNDVI, SAVI
Сельское хозяйство
B5, B6, B7, B8A, B11
Индексы: NDVI, GNDVI, NDMI, SAVI
Влажность почвы и водный стресс
B8, B11, B12
Индексы: NDMI, NBR
Пожары и выгорание
B8, B11, B12
Индексы: NBR, NDMI
Вода
B2, B3, B8, B11
Индексы: NDWI, MNDWI (если используешь SWIR)
Застройка и урбанистика
B11, B12, B8
Индексы: NDBI
   ```
### 5. Практический минимум для большинства проектов

Если нужен универсальный набор, обычно хватает:

```
каналы: B2, B3, B4, B8, B11, B12, B5, B6, B7, B8A
пресеты: 4-3-2, 8-4-3, 8-5-4, 12-11-4
индексы: NDVI, EVI, SAVI, GNDVI, NDMI, NBR, NDWI, NDBI
обязательно: cloud mask / Scene Classification Layer
```

### 6. Краткий набор формул для калькулятора индексов
```
NDVI  = (B8 - B4) / (B8 + B4)
EVI   = 2.5 * (B8 - B4) / (B8 + 6*B4 - 7.5*B2 + 1)
SAVI  = (1 + 0.5) * (B8 - B4) / (B8 + B4 + 0.5)
GNDVI = (B8 - B3) / (B8 + B3)
NDMI  = (B8 - B11) / (B8 + B11)
NBR   = (B8 - B12) / (B8 + B12)
NDSI  = (B3 - B11) / (B3 + B11)
NDBI  = (B11 - B8) / (B11 + B8)
NDWI  = (B3 - B8) / (B3 + B8)
```
