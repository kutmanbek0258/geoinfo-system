# План реализации отображения и редактирования стилей на фронтенде

## 1. Обновление моделей данных (types/api.ts)
Добавить поле `characteristics` для всех типов геообъектов, где оно отсутствует:
- `ProjectMultiline`
- `ProjectPolygon`

## 2. Утилита парсинга стилей (src/util/style.util.ts)
Создать утилиту для конвертации JSON-описания стиля из `characteristics.style` в объекты `ol/style/Style` OpenLayers.

### Функциональность:
- **Цвета**: Конвертация HEX/RGBA (включая KML-формат `#rrggbbaa`) в формат, поддерживаемый OpenLayers.
- **Точки**: 
    - Если есть `icon.url`, создавать `ol/style/Icon`.
    - Если нет, использовать стандартный `ol/style/Circle`.
    - Поддержка `scale` и `anchor`.
- **Линии**: Настройка `ol/style/Stroke` (цвет, ширина).
- **Полигоны**: Настройка `ol/style/Fill` (цвет заливки) и `ol/style/Stroke` (обводка).
- **Текст**: Поддержка `ol/style/Text` для отображения имен объектов на карте.

## 3. Отображение на карте (MapComponent.vue)
Обновить метод `updateVectorSource`:
- При создании `ol/Feature` вызывать утилиту парсинга стиля.
- Применять стиль через `feature.setStyle(style)`.
- Добавить обработчик события обновления объекта в Store для мгновенной перерисовки стиля на карте без полной перезагрузки всех объектов.

## 4. Редактирование стилей (ObjectDetails.vue)
Добавить интерфейс для редактирования стилей в диалог редактирования объекта (`editDialog`).

### UI Компоненты:
- **Выбор цвета**: Использование `v-color-picker` для выбора цвета линий и заливки.
- **Толщина линии**: Ползунок (`v-slider`) для настройки `width`.
- **Управление иконками (для точек)**:
    - Отображение текущей иконки.
    - Кнопка "Загрузить новую иконку" (интеграция с `document-service`).
    - После загрузки иконки в Minio, обновлять `characteristics.style.icon.url`.

## 5. Сохранение изменений
Обновить метод `confirmEdit`:
- Убедиться, что изменения стиля корректно упаковываются в `characteristics.style`.
- Вызывать `geodata/updateFeature`, который отправит изменения на бэкенд.

## 6. Этапы реализации
1. **Этап 1**: Создание утилиты `style.util.ts` и применение базовых стилей в `MapComponent.vue`.
2. **Этап 2**: Добавление выбора цветов и толщины линий в `ObjectDetails.vue`.
3. **Этап 3**: Реализация загрузки и применения кастомных иконок для точек.
4. **Этап 4**: Тестирование совместимости со стилями, импортированными из KML.
