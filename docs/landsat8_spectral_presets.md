## 2. Часто используемые RGB-пресеты

### Natural Color
**RGB:** `B4 - B3 - B2`

Стандартное отображение поверхности Земли в естественных цветах.

---

### False Color Vegetation
**RGB:** `B5 - B4 - B3`

Растительность отображается красным цветом.  
Используется для анализа состояния растений и сельского хозяйства.

---

### Agriculture / Vegetation Health
**RGB:** `B6 - B5 - B2`

Подходит для оценки влажности и здоровья растений.

---

### Moisture / Burn Analysis
**RGB:** `B7 - B6 - B4`

Хорошо показывает:
- выгоревшие зоны
- сухие территории
- деградацию почв

---

### Geology / Soil
**RGB:** `B7 - B5 - B3`

Используется для:
- анализа почв
- геологии
- минералогии
- различия типов поверхности

---

### Urban Analysis
**RGB:** `B7 - B6 - B2`

Подходит для:
- застройки
- инфраструктуры
- урбанизированных территорий

---

### Coastal / Water
**RGB:** `B1 - B2 - B3`

Для:
- мелководья
- береговой линии
- аэрозолей
- дымки

---

## 3. Основные индексы Landsat 8
```text
NDVI — индекс растительности
NDVI = (B5 - B4) / (B5 + B4)

EVI — улучшенный индекс растительности
EVI = 2.5 * (B5 - B4) / (B5 + 6*B4 - 7.5*B2 + 1)

SAVI — индекс для редкой растительности
SAVI = ((B5 - B4) / (B5 + B4 + 0.5)) * 1.5

MSAVI — уменьшение влияния почвы
MSAVI = (2*B5 + 1 - sqrt((2*B5 + 1)^2 - 8*(B5 - B4))) / 2

NDMI — влажность растительности
NDMI = (B5 - B6) / (B5 + B6)

NBR — выгоревшие территории
NBR = (B5 - B7) / (B5 + B7)

NBR2 — влажность после пожаров
NBR2 = (B6 - B7) / (B6 + B7)

NDSI — снег и лёд
NDSI = (B3 - B6) / (B3 + B6)

NDWI — открытая вода
NDWI = (B3 - B5) / (B3 + B5)
```
