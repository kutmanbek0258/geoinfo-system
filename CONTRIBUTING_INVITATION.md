# 🌍 Join GeoInfoSystem: Строим будущее Open-Source ГИС вместе!

Вы увлекаетесь картографией, спутниковыми снимками или высоконагруженными микросервисами? Мы ищем единомышленников для развития **GeoInfoSystem** — мощной экосистемы для сбора, анализа и визуализации геопространственных данных.

## 🚀 О проекте

**GeoInfoSystem** — это не просто карта. Это масштабируемая платформа, объединяющая возможности современных ГИС и корпоративных систем управления контентом. Мы автоматизируем путь от «сырых» спутниковых данных до интерактивной аналитики в браузере.

### Что мы уже реализовали:
*   **Микросервисная архитектура:** Spring Boot 3, Cloud Config, Eureka и API Gateway.
*   **Спутниковый конвейер:** Автоматическая обработка снимков Sentinel-2 и Landsat 8 (COG, расчет NDVI/NDWI/NDBI и др.) на Python + GDAL.
*   **3D Рельеф:** Генерация `quantized-mesh` (Cesium-ready) на лету.
*   **Real-time Video:** Стриминг с IP-камер через MediaMTX.
*   **Интеграция документов:** Полноценная работа с OnlyOffice для редактирования отчетов прямо в интерфейсе карты.
*   **Умный поиск:** Асинхронная индексация через Kafka и поиск в Elasticsearch.

## 🛠 Наш стек
*   **Backend:** Java 21, Spring Boot 3, Spring Security OAuth2.
*   **Data Processing:** Python 3.11, GDAL, Rasterio, NumPy.
*   **Frontend:** Vue.js 3, TypeScript, OpenLayers, CesiumJS.
*   **Infrastructure:** Docker Compose, Kafka, PostGIS, Elasticsearch, MinIO, GeoServer.

## 🤝 Кого мы ищем?

Проект открыт для вклада любого уровня. Нам особенно нужны:
1.  **Java-разработчики:** Развитие бизнес-логики, оптимизация API Gateway и интеграция новых микросервисов.
2.  **Python & GIS Wizards:** Улучшение алгоритмов обработки ДЗЗ, работа с облаками точек и новыми спутниковыми группировками.
3.  **Frontend-инженеры:** Создание продвинутых инструментов редактирования карт, 3D-визуализации и UX-оптимизация.
4.  **DevOps & Security:** Переход на Kubernetes, CI/CD пайплайны и укрепление безопасности OAuth2.
5.  **Data Scientists:** Внедрение ML-моделей для классификации объектов на снимках.

## 💎 Почему это круто?
*   **Сложные задачи:** Мы работаем на стыке классического веба и тяжелых геопространственных вычислений.
*   **Современный стек:** Никакого legacy — только актуальные версии фреймворков и библиотек.
*   **Вклад в индустрию:** Ваш код поможет создать доступный и мощный инструмент для экологов, градостроителей и инженеров.

## 🏁 С чего начать?
1.  Склонируйте репозиторий.
2.  Запустите стек одной командой через наш обновленный `docker-compose`.
3.  Загляните в `docs/TECHNICAL_TASK.md` и выберите задачу по душе (или предложите свою!).

**Давайте сделаем геоданные доступными и понятными вместе!**
